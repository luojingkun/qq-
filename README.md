# 网站状态检测机器人
适配 NapCat + Linux 环境，支持网页后台管理，QQ 指令检测网站在线状态

## 开发者

## 支持功能
- 指令触发网站状态检测
- 响应时间、状态码、连通性检测
- 可视化网页后台管理所有配置
- 支持批量添加网址
- 适配 Linux 全版本

## Linux 部署教程
1. 上传项目到服务器
2. 赋予启动权限：chmod +x start.sh
3. 运行：./start.sh
4. 访问后台：http://服务器IP:2025
5. 配置完成后重启机器人即可使用

## 指令
默认：状态
可在后台自定义修改
