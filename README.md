# 网站状态检测机器人
基于 NapCat 开发的 QQ 网站状态检测机器人，支持可视化后台配置管理

## 开发者
灵烁
邮箱：lingshuo070330@163.com

## 功能说明
1. 发送指令自动检测网站在线状态
2. 可视化后台管理所有配置
3. 支持批量添加检测网址
4. 实时响应时间、状态码展示
5. 适配所有 NapCat 环境

## 使用方法
1. 安装依赖：pip install -r requirements.txt
2. 启动脚本：python main.py
3. 访问后台：http://127.0.0.1:2025
4. 配置完成后重启脚本即可使用

## 指令说明
默认指令：状态
可在后台自定义修改指令内容
